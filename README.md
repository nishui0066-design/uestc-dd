# 校搭 (Campus Match) - 校园运动搭子平台

## 📖 项目简介

校搭是一个面向校园用户的运动搭子匹配平台，支持多人联机、实时聊天、约局组队、群聊等功能。用户可以根据运动偏好、段位找到志同道合的运动伙伴。

## ✨ 功能特性

### 核心功能
- 🏠 **广场** - 在线用户浏览、筛选、智能匹配、邀请搭子
- 📢 **约局大厅** - 发起约局、报名参加、退出约局、取消约局
- 👥 **群聊** - 创建兴趣群组、群内实时聊天、加入/退出群聊
- 💬 **私聊** - 与搭子一对一实时聊天
- 👤 **个人中心** - 个人信息管理、搭子管理、活动记录

### 技术亮点
- 📱 响应式设计，手机/电脑自适应
- 🔄 实时数据同步（3秒轮询）
- 🧠 智能匹配算法（运动、段位、状态综合评分）
- 📦 PWA 支持，可安装到手机桌面

## 🛠️ 技术栈

| 技术 | 用途 |
|------|------|
| Node.js + Express | 后端服务 |
| 原生 HTML/CSS/JS | 前端页面 |
| JSON 文件 | 数据存储 |
